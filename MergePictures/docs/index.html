MergePictures — User Guide

Basic Settings

  • Merge Count: Number of images to combine per result.
  • Direction: Vertical (top-to-bottom) or Horizontal (side-by-side).
  • Preview Quality: Low / Medium / High — adjusts preview speed and clarity (does not affect final export quality).
  • Uniform Scaling: Toggle proportional scaling on/off. When on, images are scaled so a chosen dimension matches before merging.
  • Uniform Dimension: Proportionally scale all images so either widths or heights match before merging.
    • Auto: Vertical merges unify widths; Horizontal merges unify heights.
    • Fit Width: Always unify widths.
    • Fit Height: Always unify heights.
  • Scale Strategy: How to choose the target size for the unified dimension. The app shows a live description matching your current selection.
    • Min: Shrink all images to the smallest width/height (no upscaling).
    • Max: Enlarge all images to the largest width/height (no downscaling).
    • Average: Scale all images to the average width/height.

基础设置

  • 合并数量：每张合成图片包含的源图数量。
  • 方向：纵向（从上到下）或横向（并排）。
  • 预览质量：低 / 中 / 高，用于调节预览速度与清晰度（不影响最终导出质量）。
  • 统一缩放：可开关。开启后，将在合并前对图片进行等比例缩放。
  • 统一维度：合并前将所有图片等比例缩放，使其宽度或高度一致。
    • 自动：纵向合并统一宽度；横向合并统一高度。
    • 统一宽度：始终统一宽度。
    • 统一高度：始终统一高度。
  • 缩放策略：选择统一维度的目标大小。界面会根据你的选择实时显示对应说明。
    • 最小：将所有图片缩小到组内最小的宽/高（不放大）。
    • 最大：将所有图片放大到组内最大的宽/高（不缩小）。
    • 平均：将所有图片缩放到组内平均的宽/高。

Tip: These settings affect both preview generation and exported results.

提示：以上设置同时影响预览与导出的合成结果。
