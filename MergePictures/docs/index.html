快速上手

  1. 导入图片：在 iOS 或 macOS 中选择多张图片。
  2. 选择布局：水平 / 垂直 / 网格。
  3. 导出设置：分辨率、自定义画布；如需可选的精确压缩，请输入目标大小（KB/MB）。
  4. 一键合并并保存至相册 / 文件。
提示：所有处理均在设备本地完成，不会上传到服务器。

常见问题（FAQ）

1) 如何获得无损或近乎无损的结果？

  • 导出格式选择 PNG 或高质量 JPEG。
  • 关闭多次重复压缩；必要时使用「可选的精确压缩」并设置足够的目标大小。

2) 结果体积与目标大小略有差异？

受图像内容与编码器限制，精确压缩会在目标值附近微调（±数 KB）。建议将目标大小设为略高于最低需求。

3) 导出太慢或内存不足？

  • 降低单次合并的图片数量或分辨率。
  • 在 macOS 上优先使用 Apple Silicon 设备。

4) 颜色偏差/显示不一致？

  • 确保素材为 sRGB；导出时也选择 sRGB。

FAQ

1) How to get lossless or near-lossless results?

  • Export as PNG or high-quality JPEG.
  • Avoid repeated re-encoding; when needed, use “optional precise compression” with a sufficient target size.

2) Why is the final size slightly off the target?

Due to image content and encoder behavior, precise compression converges near the target (±a few KB). Set a slightly higher target if required.

3) Slow export or memory pressure?

  • Reduce image count per batch or lower resolution.
  • Use Apple Silicon Macs for best performance.

4) Color mismatch?

  • Ensure source images are sRGB; export as sRGB as well.
